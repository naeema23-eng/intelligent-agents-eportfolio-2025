Overview

This e-Portfolio collects my key activities, discussions and reflections for the Intelligent Agents module. Each HTML page corresponds to a specific activity listed in the module handbook, including collaborative discussions, seminar preparation tasks and practical exercises.

My aim is to document how my understanding of agent architectures, communication languages and deep learning evolved over the twelve units, and how these ideas connect back to real-world applications in healthcare, logistics and autonomous decision-making.
